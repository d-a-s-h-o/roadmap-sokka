The Sokka Project Roadmap

The Code Depot

This is a SSH git server that will be used to store all of our code. We plan to add our stable releases to this server and make it available to the public.

Released

The Website

We're revamping our website to make it more user friendly and easier to navigate. We're also adding a blog to the website so that we can keep our users up to date on the latest news.

Planned for release in 2022

The Documentation

We're working on a new documentation website that will be used to document all of our projects. It will be easier to use, and will be more organized than our current documentation.

Planned for release in 2022

Dashed Droplets v5

Okay okay okay, we know that we've been promising this for a while now, but we're finally working on a new version of Dashed Droplets. It will be faster, more secure, and easier to use than ever before. It will also be open source and free to use, and packed with a ton of new features. Seriously, this release is going to be awesome.

Planned for release in 2022

A SSH Marketplace

Yes, you read that right. We're working on a marketplace accessible via SSH that will allow users to buy and sell goods and services right from their terminal. It's safer than using a web browser, and a whole lot faster. We're also working on a way to allow users to pay for goods and services using cryptocurrency like Monero and Zcash.

Planned for release in 2023

A New Forum

We're working on an all new forum that will make sharing information easier than ever. Open source and free to use, it will be a great tool for anyone hoping to set up a community. Like honestly though, who doesn't love a good forum?

Planned for release in 2023

Verification Made Easy

With a steady increase in the number of people pretending to be us, we're working on a way to verify that you're talking to the real Dasho, such as the verify.dasho.dev utility.

Planned for release in 2023

A News Platform

Imagine a news platform that's open source, free to use, and accessible via SSH. Well, we're working on it. A quick and easy way to stay up to date on the latest news, from a source you can trust, built on technology you can trust.

Planned for release in 2023

The Everglade

This is sort of a secret, so don't tell anyone, but we're redefining the way people connect to the internet. I can't say much more than that, but it's going to be good.

Planned for release in 2024

Open Source Everything

So you want to set up a chat room huh? A website, a forum, a blog, a marketplace, a VPN, a mail server, your own version of netflix, a couple of darkweb sites, and even a file host? Yep, uploading the source code for you now ...

Planned for release in 2024

DashOS

We're working on a new operating system that will be open source and free to use. It will be based on Linux, and will be designed to be easy to use, secure, and fast. It will also be packed with a ton of features, and will be available for a wide variety of devices. It will be especially useful for those who want to host various services on their own hardware, such as a web server, a mail server, a VPN, a file host, and more. Alongside the operating system, we're also working on a new package manager for DashOS that will make installing and updating these services (or applets) a breeze.

Planned for release in 2025
